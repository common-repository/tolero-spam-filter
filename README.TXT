== Description ==
Spam filer works by checking a user behavior via webserver log file.

== Benefits ==
* No disturbing users captcha, e-mail verification, and dubious content heuristics
* Easy installation: just copy one file and activate this wordpress plugin
* Invisible to the spammers

== Requirements ==
* Access to the log file	
  Log should be in a default apache format.
  It should contain at least a two last days data (logrotate is supported).
* Wordpress version 1.5.1.2 or later

== Installation ==

1. Download and unzip a plugin
2. Put the tolero_filter.php into wp-content/plugins subdirectory of your blog
3. At the administrator's controls, go to the plugins and activate the Tolero blog spam filter
4. Open the Tolero filter Configuration tab and fill the path to the log file
